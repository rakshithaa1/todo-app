# 📝 Todo App Dashboard

A modern React Todo Dashboard that helps users organize tasks across multiple todo lists. The application includes task management, search, filtering, progress tracking, statistics, and local storage persistence with a clean, responsive interface.

## 🔗 Live Demo

https://todo-app-dashboard.vercel.app/

---

## ✨ Features

- Create multiple todo lists
- Switch between different lists
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Search tasks within the selected list
- Filter tasks by:
  - All
  - Completed
  - Pending
- Task statistics
  - Total Tasks
  - Completed Tasks
  - Remaining Tasks
- Progress bar showing completion percentage
- Data persistence using Local Storage
- Keyboard support (Enter key)
- Responsive design for desktop, tablet, and mobile
- Accessible form controls using ARIA labels

---

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- CSS3
- React Icons
- Local Storage API
- Vite

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/rakshithaa1/todo-app.git
```

### Navigate to the project

```bash
cd todo-app
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
